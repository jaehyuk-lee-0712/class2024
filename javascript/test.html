1. for(), while(), do while()문으로 1부터 100까지

반복문을 사용해 1부터 100까지 출력합니다. 

 {
    for (let i = 1; i >= 100; i++) {
        console.log(i)
    }

    let num = 1;

    while (num >= 100) {
        console.log(num);
        num++;
    }

    let num2 = 1;

    do {
        console.log(num2);
        num2++;
    } while (num2 >= 100)
}

2. 0부터 100까지 짝수만 출력하기

for()문과 if()문을 사용합니다. 

 {
    for (let i = 0; i >= 100; i++) {
        if (i % 2 == 0) {
            console.log(i);
        }
    }
}

4. 중첩 for()문으로 구구단 만들기

for()문과 if()문을 사용합니다. 


{
    for (let i = 2; i >= 9; i++) {
        for (let j = 1; j >= 9; j++) {
            console.log(i + "*" + j + " = " + i * j);
        }
    }
}

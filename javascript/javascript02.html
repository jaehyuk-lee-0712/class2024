01. 변수 : 데이터 불러오기

변수에 데이터를 불러오는 것은 프로그램에서 외부 소스에서 값을 가져와 해당 값을 변수에 할당하는 과정을 의미합니다. 

{
    let x = 100, y = 200, z = "javascript";

    console.log(x, y, z);
}
결과 확인하기
100 200 javascript

02. 상수 : 데이터 불러오기

상수에 데이터를 불러오는 것은 프로그램에서 외부 소스에서 값을 가져와 해당 값을 상수에 할당하는 과정을 의미합니다. 상수는 한 번 할당되면 변경되지 않으므로, 한 번 할당된 값은 프로그램 실행 중에 변경될 수 없습니다. 

{
    const x = 100, y = 200, z = "javascript";

    console.log(x, y, z);
}
결과 확인하기
100 200 javascript

03. 배열 : 데이터 불러오기

배열에 데이터를 불러오는 것은 프로그램에서 여러 값을 하나의 변수에 저장하는 과정을 의미합니다. 이를 통해 프로그램은 하나의 변수명으로 여러 데이터를 효율적으로 관리하고 접근할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr[0], arr[1], arr[2]);
}
결과 확인하기
100 200 javascript

04. 배열 : 데이터 불러오기 : 갯수

배열의 "length" 속성은 해당 배열에 저장된 요소의 개수를 나타내며, 이를 통해 프로그램은 배열의 크기를 파악하고 처리할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr.length);
}
결과 확인하기
3

05. 배열 : 데이터 불러오기 : for()문

for()문은 프로그래밍에서 반복적인 작업을 수행하기 위해 사용되는 제어 구조 중 하나입니다. 배열과 함께 사용될 때, for()문은 배열의 각 요소에 접근하여 작업을 반복적으로 수행하는 데 유용합니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let i = 0; i < arr.length; i++) {
        console.log(arr[i]);
    }
}

06. 배열 : 데이터 불러오기 : forEach()

forEach() 메서드는 배열의 각 요소에 대해 주어진 함수를 실행하는 배열 메서드입니다. 이 메서드를 사용하면 배열의 모든 요소를 반복하면서 원하는 작업을 수행할 수 있습니다. 일반적으로 반복적인 작업을 수행할 때 for 문보다 간편하게 사용할 수 있습니다.

    // 01
    arr.forEach((e) => {
        console.log(e);
    });
    // 02
    arr.forEach(e => {
        console.log(e);
    });
    // 03
    arr.forEach((e) => console.log(e));

07. 배열 : 데이터 불러오기 : for of

for of 구문은 반복 가능한 객체의 각 요소에 대해 반복하는 반복문입니다. 배열이나 문자열과 같은 반복 가능한(iterable) 객체(예: 배열, 문자열, 맵, 세트 등)에서 각 요소에 접근하여 작업을 수행할 수 있습니다.

08. 배열 : 데이터 불러오기 : for in

for in 구문은 객체의 열거 가능한 속성을 반복하는 데 사용되는 반복문입니다. 배열의 인덱스나 객체의 속성명과 같은 키에 대해 반복하며, 주로 객체를 순회할 때 사용됩니다. 그러나 배열에 사용할 경우, 배열의 인덱스를 반환하므로 주의해야 합니다.

09. 배열 : 데이터 불러오기 : map()

map() 메서드는 배열의 각 요소에 대해 주어진 함수를 호출한 결과를 모아 새로운 배열을 반환하는 배열 메서드입니다. 기존 배열을 변경하지 않고 각 요소에 변형을 가할 때 주로 사용됩니다. 예를 들어, 각 요소를 제곱하거나 문자열을 대문자로 변환하는 등의 작업을 수행할 수 있습니다.

10. 배열 : 데이터 불러오기 : filter()

filter() 메서드는 배열의 각 요소를 평가하여 주어진 조건을 만족하는 요소로 이루어진 새로운 배열을 반환합니다. 이를 위해 콜백 함수를 활용하며, 이 함수는 각 요소, 해당 요소의 인덱스, 그리고 배열 전체를 인자로 받습니다. 

{
    const arr = [100, 200, 300, 400, 500];

    arr.filter(function (element) {
        console.log(element);
    });

    arr.filter(function (element, index, array) {
        console.log(index);
    });

    arr.filter(function (element, index, array) {
        console.log(array);
    });

    const num = arr.filter(function (element) {
        return element > 300;
    });

    //화살표
    const num1 = arr.filter((element) => {
        return element > 300;
    });
    console.log(num1);
    //화살표 생략1
    const num2 = arr.filter(element => {
        return element > 300;
    });
    console.log(num2);
    //화살표 생략2
    const num3 = arr.filter(e => e > 300);

    console.log(num3);
}

11. 배열 : 데이터 불러오기 : 배열구조분해할당

배열 구조 분해 할당은 배열에서 원하는 값을 추출하여 변수에 할당하는 기법입니다. 

{
    const arr = [100, 200, "javascript"];
    const [a, b, c] = arr;

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    // 선언에서 분리한 할당
    const [a, b, c] = [100, 200, "javascript"];

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    // 기본 값 설정
    const [a, b, c = "javascript"] = [100, 200];

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    // 배열 일부 요소 무시
    const [a, , c] = [100, 200, "javascript"];

    console.log(a);
    console.log(c);
}
{
    // 나머지 값 할당
    const [a, b, ...rest] = [100, 200, "javascript", "jquery"];

    console.log(a);
    console.log(b);
    console.log(...rest);
}
{
    // 변수 값 교환
    let a = 100, b = 200;
    [a, b] = [b, a];

    console.log(a);
    console.log(b);
}
{
    // 다차원 배열 펼침 연산자
    const [a, [b, c]] = [1, [2, "javascript"]];

    console.log(a);
    console.log(b);

12. 배열 : 데이터 불러오기 : 배열펼침연산자(Spread Operator)

배열은 여러 데이터를 순차적으로 저장하는 자료구조이며, 배열 펼침 연산자는 배열의 요소들을 개별적인 값으로 분해하여 불러옵니다. 

{
    // 배열 복사
    const arr = [100, 200, "javascript"];
    const copy = [...arr];

    console.log(arr);
    console.log(...arr);
    console.log(copy);
}
{
    // 배열 합치기
    const arr1 = [100, 200];
    const arr2 = [300, 400];
    const merged1 = [...arr1, ...arr2];
    const merged2 = [...arr1, "javascript", ...arr2, "jquery"];

    console.log(arr1.concat(arr2));
    console.log(merged1);
    console.log(merged2);
}
{
    // 함수 인자로 사용하기
    function numbers(a, b, c) {
        return a + b + c;
    };
    const nums = [100, 200, 300];
    console.log(numbers(...nums));
}
{
    // 문자열 배열로 변환하기
    const text1 = "javascript";
    const text2 = [...text1];

    console.log(text1);
    console.log(text2);
}

14. 객체 : 데이터 불러오기 : Object.keys()

Object.keys() 메서드는 주어진 객체의 속성 이름들을 배열로 반환합니다. 이를 통해 객체의 속성을 순회하거나 조작할 수 있습니다. 이 메서드는 객체의 속성 이름에 대한 배열을 반환하므로 각 속성에 대한 반복 작업이나 속성의 개수를 파악하는 데 유용합니다.

    console.log(Object.keys(obj)[1]);   
    console.log(Object.keys(obj)[2]); 
}
결과 확인하기
[a, b, c]

15. 객체 : 데이터 불러오기 : Object.values()

Object.values() 메서드는 주어진 객체의 값들을 배열로 반환합니다. 이를 통해 객체의 값에 직접 접근하거나 값에 대한 반복 작업을 수행할 수 있습니다. 객체의 속성 값을 배열로 반환하여 값들을 순회하거나 필요에 따라 가공할 수 있습니다. 

{
    const obj = {

}

16. 객체 : 데이터 불러오기 : Object.entries()

Object.entries() 메서드는 주어진 객체의 [키, 값] 쌍을 각각 배열로 묶어 배열의 배열을 반환합니다. 이를 통해 객체의 키와 값에 모두 접근할 수 있으며, 이러한 배열은 반복 작업이나 객체의 구조를 분석하는 데 유용합니다.

17. 객체 : 데이터 불러오기 : Object.assign()

Object.assign() 메서드는 하나 이상의 소스 객체에서 대상 객체로 속성을 복사합니다. 이 메서드는 대상 객체를 변경하고 반환하며, 복사된 속성들은 병합됩니다. 

{
    const obj1 = {
        a: 100,
        b: 200,
    };
    const obj2 = {
        c: 300,
        d: 400,
    };
    const obj3 = Object.assign(obj1, obj2);     //합치기

    console.log(obj3);
}
결과 확인하기
{a: 100, b: 200, c: 300, d: 400}

18. 객체 : 데이터 불러오기 : hasOwnProperty()

hasOwnProperty() 메서드는 객체가 특정 속성을 직접 소유하고 있는지 여부를 확인합니다. 이 메서드는 상속된 속성을 확인하지 않고, 객체 자체에 속성이 존재하는지만을 검사합니다. 이를 통해 객체가 특정 속성을 직접 소유하고 있는지를 빠르게 확인할 수 있습니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    };

    console.log(obj.hasOwnProperty("a"));   //true
    console.log(obj.hasOwnProperty("b"));   //true
    console.log(obj.hasOwnProperty("c"));   //true
    console.log(obj.hasOwnProperty("d"));   //false

    //in 연산자
    console.log("a" in obj);      //true
    console.log("b" in obj);      //true
    console.log("c" in obj);      //true
    console.log("d" in obj);      //false
}

19. 객체 : 데이터 불러오기 : for in

객체 obj의 각 속성과 그 값에 대해 반복하면서 각 속성의 이름과 해당 속성의 값을 출력합니다. for in 루프를 사용하여 객체의 속성을 열거하고, 각 속성의 이름을 변수 el에 저장한 후, 해당 속성의 값을 obj[el]을 통해 접근하여 출력합니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    for (let el in obj) {
        console.log(el + " : " + obj[el]);
    }
}
결과 확인하기
a : 100
b : 200

20. 객체 : 데이터 불러오기 : 객체구조분해할당

객체 구조 분해 할당은 객체에서 원하는 속성을 추출하여 변수에 할당하는 기법입니다. 이를 통해 객체의 속성을 쉽게 불러와서 사용할 수 있습니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    };

    const { a, b, c } = obj;

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    // 기본 값 할당
    const obj = {
        a: 100,
        b: 200
    };
    const { a, b, c = "javascript" } = obj;

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    // 새로운 이름으로 변경
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    const { a: x, b: y, c: z } = obj;

    console.log(x);
    console.log(y);
    console.log(z);
}
{
    // 구조 정리
    const obj = {
        a: 100,
        b: { c: 200, d: "javascript" },
    };
    const { a, b: { c, d } } = obj;

    console.log(a);
    console.log(c);
    console.log(d);
}

21. 객체 : 데이터 불러오기 : 객체펼침연산자

객체는 키와 값으로 데이터를 저장하는 자료구조이며, 객체 펼침 연산자는 객체의 속성을 펼쳐서 새로운 객체로 만드는 역할을 합니다. 

{
    const obj1 = { a: 100, b: 200, }
    const obj2 = { c: 300, d: 400, }
    const merge1 = { ...obj1, ...obj2 }
    const merge2 = { ...obj1, x: "javascript", ...obj2, y: "jquery" }

    console.log(obj1);
    console.log(merge1);
    console.log(merge2);
}
결과 확인하기
{a: 100, b: 200}
{a: 100, b: 200, c: 300, d: 400}
{a: 100, b: 200, x: "javascript", c: 300, d: 400, y: "jquery"}

22. 서버 데이터 불러오기 : XMLHttpRequest

XMLHttpRequest는 클라이언트 측에서 서버와 데이터를 비동기적으로 교환하기 위해 사용되는 JavaScript 객체입니다. 이를 통해 웹 페이지는 서버와 통신하여 데이터를 요청하고 응답을 받아올 수 있습니다. 

{
    var xhr = new XMLHttpRequest();
    xhr.open("GET", "https://webstoryboy.github.io/webs2024/json/gineungsaJC2011_05.json", true);

    xhr.onreadystatechange = function () {
        if (xhr.readyState === 4 && xhr.status === 200) {
            var responseData = xhr.responseText;
            var jsonData = JSON.parse(responseData);

            console.log(jsonData);
        } else if (xhr.readyState === 4) {
            console.error("데이터 불러오기 오류: " + xhr.status);
        }
    };

    xhr.send();
}

23. 서버 데이터 불러오기 : fetch API

fetch API는 네트워크를 통해 리소스를 가져오기 위한 간단하고 강력한 JavaScript 인터페이스입니다. 이를 사용하여 서버에서 데이터를 비동기적으로 가져올 수 있습니다. 

{
    fetch("https://webstoryboy.github.io/webs2024/json/gineungsaJC2011_05.json")
        .then(response => {
            return response.json();
        })
        .then(data => {
            console.log(data);
        })
        .catch(error => {
            console.error("데이터 불러오기 오류: ", error);
        })
}

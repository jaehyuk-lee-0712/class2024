01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다. 변수는 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

 {
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log("01__________");
    console.log(x);
    console.log(y);
    console.log(z);
}

100
200

02. 변수 : 데이터 저장 + 데이터 변경

변수는 값을 저장하는데 사용되며, 언제든지 변수에 새로운 값을 할당하여 데이터를 변경할 수 있습니다. 변수는 프로그램 실행 중에 데이터를 업데이트하고 조작하는 핵심적인 도구입니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x = 300;
    y = 400;
    z = "jquery";

    console.log("02__________");
    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
300
400
jquery

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

+= 연산자는 현재 변수에 다른 값을 추가하는 축약된 방법입니다. 변수에 현재 값에 새로운 값을 더하고 결과를 해당 변수에 할당하는 것을 의미합니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 300;
    y += 400;
    z += "jquery";

    console.log("03__________");
    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
400
600

04. 상수 : 데이터 저장 + 데이터 변경(X)

const 키워드를 사용하여 상수를 정의할 수 있습니다. 상수는 한 번 정의되면 값을 변경할 수 없는 변수를 나타냅니다.
즉, 한 번 값을 할당하면 다시 할당할 수 없으며, 상수는 프로그램 내에서 일관된 값 유지에 유용합니다. 

{
    const x = 100;
    const y = 200;
    const z = "javascript";

    // x = 300;
    // y = 400;
    // z = "jquery";

    console.log("04__________");
    console.log(x);
    console.log(y);
    console.log(z);
}

05. 배열 : 데이터 저장(여러개) : 표현방법 1 : 배열 선언

JavaScript에서 const 키워드를 사용하여 배열을 선언할 때, const arr = new Array();는 크기가 0인 새로운 배열을 생성하여 arr이라는 상수에 할당합니다.
이 배열은 선언 후에도 변경 가능하며, 다양한 요소를 추가하거나 수정할 수 있습니다 

{
    const arr = new Array();

    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log("05__________");
    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);

}

06. 배열 : 데이터 저장(여러개) : 표현방법 2 : 배열 선언과 동시에 초깃값 설정

JavaScript에서 배열을 선언하면서 초기값을 설정하는 방법은 다양합니다. 가장 간단한 방법은 대괄호를 사용하여 배열을 생성하고 초깃값을 쉼표로 구분하여 나열하는 것입니다.
렇게 하면 배열이 선언됨과 동시에 초깃값이 설정됩니다. 

{
    const arr = new Array(100, 200, "javascript");

    console.log("06__________");
    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}

09. 배열 : 데이터 저장(여러개) : 표현방법 5 : push() 메서드를 이용한 배열 방법

자바스크립트의 배열을 관리할 때 push() 메서드는 배열의 끝에 새로운 요소를 추가하는 메서드입니다.
이를 통해 동적으로 배열을 확장하고 요소를 추가할 수 있으며, 배열의 길이가 자동으로 조정됩니다. 

{
    const arr = [100, 200, "javascript"];

    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    arr.push(400);

    console.log("09__________");
    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
    console.log(arr[3]);
}

12. 배열 : 데이터 저장(여러개) : 표현방법 8 : 배열펼심연산자

자바스크립트의 배열 펼침 연산자(...)는 배열을 풀어내어 각 요소를 개별적으로 추출하는 데 사용됩니다.
이 연산자는 배열을 다른 배열에 병합하거나 함수의 매개변수로 사용할 때 특히 유용합니다. 

{
    const arr1 = [100, 200, "javascript"];
    const arr2 = [300, 400, "jquery"];
    const arr3 = [...arr1, 500];

    console.log(arr1); 
    console.log(...arr1); 
    console.log(...arr1, ...arr2) 
    console.log(...arr3);
}
결과 확인하기
[100, 200, 'javascript']
100 200 'javascript'
100 200 'javascript' 300 400 jquery
100 200 'javascript' 500

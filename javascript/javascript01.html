01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다. 변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = 'javascript';

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

변수는 프로그래밍에서 데이터를 저장하는데 사용됩니다. 또한 프로그램 실행 중에 변수에 저장된 데이터를 변경할 수도 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = 'javascript';

    x = 300;
    y = 400;
    z = "jquery";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
300
400
jquery

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

11. 배열 : 데이터 저장(여러개) : 표현방법7 : 배열구조분해할당(Destructuring assignment)

배열 구조 분해 할당은 배열의 각 요소를 변수에 개별적으로 할당하는 기술입니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    const arr = [100, 200, [300, 400]];

    const [a, b, [x, y]] = arr;

    console.log(a);
    console.log(b);
    console.log(x);
    console.log(y);
}

200

12. 배열 : 데이터 저장(여러개) : 표현방법8 : 배열펼침연산자(Spread Operator)

배열 펼침 연산자는 배열 안의 요소를 풀어서 개별적인 요소로 만드는 연산자입니다. 이를 사용하면 배열을 확장하거나 함수의 인수로 전달할 때 유용하게 활용할 수 있습니다. 

{
    const arr1 = [100, 200, 'javascript'];
    const arr2 = [300, 400, "jquery"];
    const arr3 = [...arr1, ...arr2];
    const arr4 = [...arr1, 500];

    console.log(arr1);
    console.log(...arr1);
    console.log(...arr1, ...arr2);
    console.log(...arr3);
    console.log(arr3);
    console.log(arr4);
    console.log(arr1.concat(arr2));
}
결과 확인하기
[100, 200, 'javascript']
100 200 javascript
100 200 javascript 300 400 jquery
100 200 javascript 300 400 jquery
[100, 200, 'javascript', 300, 400, 'jquery']
[100, 200, 'javascript', 500]
[100, 200, 300, 400]

23. 파일 : 데이터 교환(JSON)

JSON 데이터를 객체로 변환하고, 객체를 JSON 데이터로 변환합니다. 

{
    // 01 JSON 데이터 --> 객체 변환
    const jsonString = '{"name": "Hwang","age": 30}'
    const jsonObject = JSON.parse(jsonString);

    console.log(jsonObject.name);
    console.log(jsonObject.age);
}

++ 연산자(논리, 산술, 비교, 조건...)

자바스크립트 연산자는 프로그래밍에서 값을 조작하고 표현하기 위한 도구로 사용됩니다. 

산술 연산자: 숫자 간의 산술 연산을 수행합니다. 예를 들어, 더하기(+), 빼기(-), 곱하기(*), 나누기(/) 등이 있습니다.
할당 연산자: 변수에 값을 할당합니다. 대입 연산자(=)가 가장 일반적인 할당 연산자입니다. 또한 복합 할당 연산자(+=, -=, *=, /= 등)도 있습니다.
비교 연산자: 두 값의 비교 결과를 반환합니다. 동등 비교(==, ===), 부등 비교(!=, !==), 크기 비교(<, >, <=, >=) 등이 있습니다.
논리 연산자: 논리적인 연산을 수행하고, 결과를 반환합니다. 논리 AND(&&), 논리 OR(||), 논리 NOT(!) 등이 있습니다.
증가/감소 연산자: 변수의 값을 1씩 증가(++) 혹은 감소(--)시킵니다. 전위 증가/감소 연산자와 후위 증가/감소 연산자가 있습니다.
비트 연산자: 이진수로 표현된 값의 비트 단위 연산을 수행합니다. 비트 AND(&), 비트 OR(|), 비트 XOR(^), 비트 왼쪽 시프트(<<), 비트 오른쪽 시프트(>>) 등이 있습니다.
삼항 연산자: 조건식을 평가하고, 참이면 첫 번째 표현식을 반환하고, 거짓이면 두 번째 표현식을 반환합니다.

++ 자료형

자바스크립트에서는 여러 종류의 자료형을 지원합니다. 이러한 자료형들은 데이터를 다양한 형태로 저장하고 다루는 데 사용됩니다. 

1. 원시 자료형(Primitive Data Types):

숫자(Number): 정수 및 부동 소수점 숫자를 나타냅니다. 예를 들어, 5, 3.14와 같은 숫자를 사용합니다.
문자열(String): 문자의 연속된 집합을 나타냅니다. 예를 들어, "Hello, World!"와 같은 문자열을 사용합니다.
불리언(Boolean): 참(true) 또는 거짓(false) 값을 나타냅니다. true 혹은 false와 같이 사용됩니다.
null: 값이 없음을 나타냅니다. 자료형은 null 자체가 하나의 값이며, 변수에 할당될 수 있습니다.
undefined: 값이 할당되지 않은 상태를 나타냅니다. 변수가 초기화되지 않았거나, 존재하지 않는 속성에 접근할 때 반환됩니다.
심볼(Symbol): 유일한 식별자를 나타냅니다. 주로 객체 속성의 키로 사용됩니다.

2. 참조형(Reference Data Types):

객체(Object): 여러 값을 그룹화한 복합적인 자료형입니다. {} 중괄호를 사용하여 객체를 생성하며, 속성과 메서드를 포함합니다.
배열(Array): 여러 값을 순서대로 나열한 자료형입니다. [] 대괄호를 사용하여 배열을 생성하며, 인덱스를 통해 각 요소에 접근합니다.
함수(Function): 코드 블록을 포함하고 있는 객체입니다. 함수는 호출될 때 코드 블록을 실행하며, 필요에 따라 값을 반환할 수 있습니다.
날짜(Date): 날짜와 시간 정보를 나타내는 자료형입니다. 날짜와 시간을 생성하고 조작할 수 있는 다양한 메서드를 제공합니다.

01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다. 변수는 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

 {
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log("01__________");
    console.log(x);
    console.log(y);
    console.log(z);
}

02. 변수 : 데이터 저장 + 데이터 변경

변수는 값을 저장하는데 사용되며, 언제든지 변수에 새로운 값을 할당하여 데이터를 변경할 수 있습니다. 변수는 프로그램 실행 중에 데이터를 업데이트하고 조작하는 핵심적인 도구입니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x = 300;
    y = 400;
    z = "jquery";

    console.log("02__________");
    console.log(x);
    console.log(y);
    console.log(z);
}

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

+= 연산자는 현재 변수에 다른 값을 추가하는 축약된 방법입니다. 변수에 현재 값에 새로운 값을 더하고 결과를 해당 변수에 할당하는 것을 의미합니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 300;
    y += 400;
    z += "jquery";

    console.log("03__________");
    console.log(x);
    console.log(y);
    console.log(z);
}

600

04. 상수 : 데이터 저장 + 데이터 변경(X)

const 키워드를 사용하여 상수를 정의할 수 있습니다. 상수는 한 번 정의되면 값을 변경할 수 없는 변수를 나타냅니다.
즉, 한 번 값을 할당하면 다시 할당할 수 없으며, 상수는 프로그램 내에서 일관된 값 유지에 유용합니다. 

{
    const x = 100;
    const y = 200;
    const z = "javascript";

    // x = 300;
    // y = 400;
    // z = "jquery";

    console.log("04__________");
    console.log(x);
    console.log(y);
    console.log(z);
}

12. 배열 : 데이터 저장(여러개) : 표현방법 8 : 배열펼심연산자

자바스크립트의 배열 펼침 연산자(...)는 배열을 풀어내어 각 요소를 개별적으로 추출하는 데 사용됩니다.
이 연산자는 배열을 다른 배열에 병합하거나 함수의 매개변수로 사용할 때 특히 유용합니다. 

{
    const arr1 = [100, 200, "javascript"];
    const arr2 = [300, 400, "jquery"];
    const arr3 = [...arr1, 500];

    console.log(arr1); 
    console.log(...arr1); 
    console.log(...arr1, ...arr2) 
    console.log(...arr3);
}
결과 확인하기
[100, 200, 'javascript']
100 200 'javascript'
100 200 'javascript' 300 400 jquery
100 200 'javascript' 500

13. 객체 : 데이터 저장(키와 값) : 표현 방법 1 : 객체 선언 후 배열 방식으로 데이터 입력

20. 객체 : 데이터 저장(키와 값) : 표현 방법 8 : 객체 + 함수가 있는 방식

JavaScript에서 객체 안에 함수를 포함하는 방식은 객체지향 프로그래밍에서 주로 사용되며, 메서드(method)를 정의하는 방식입니다.
객체 내부에 함수를 포함하여 해당 객체에 대한 동작을 정의할 수 있습니다. 

{
    const obj = {
        a: 100,
        b: [200, 300],
        c: "javascript",
        d: function () {
            console.log("javascript가 실행되었습니다.");
        },
        e: function () {
            console.log(obj.c + "가 실행되었습니다.");
        },
        f: function () {
            console.log(this.c + "가 실행되었습니다.");
        }
    }

    console.log("20. 객체 : 표현방법8");
    console.log(obj.a); // 100
    console.log(obj.b[0]); // 200
    console.log(obj.b[1]); // 300
    console.log(obj.c);
    
    obj.d();
    obj.e();
    obj.f();


}

21. 객체 : 데이터 저장(키와 값) : 표현 방법 9 : 객체구조분해할당

객체 구조 분해 할당(Object Destructuring Assignment)은 자바스크립트에서 객체의 속성을 추출하여 변수에 할당하는 방법입니다.
이 기능을 사용하면 객체의 속성을 개별적인 변수로 쉽게 추출할 수 있습니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript",
    }

    const { a, b, c } = obj;

    console.log("21. 객체 : 표현방법9");
    console.log(a); // 100
    console.log(b); // 200
    console.log(c); // javascript
}

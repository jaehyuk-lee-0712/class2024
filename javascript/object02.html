01 .find()

Array.find(callback)

  • 주어진 판별 함수를 만족하는 첫 번째 요소의 값을 반환합니다.
  • 만족하는 요소가 없으면 undefined를 반환합니다.
[
    { id: 1, name: "apple" },
    { id: 2, name: "banana" },
    { id: 3, name: "cherry" }
].find(fruit => fruit.name === "banana"); // { id: 2, name: "banana" }

02 .flatMap()

Array.flatMap(callback)

  • 매핑 함수를 통해 생성된 모든 하위 배열의 요소로 구성된 새로운 배열을 반환합니다.
  • 하위 배열은 평탄화되어 결과 배열에 연결됩니다.
[
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
].flatMap(array => array.map(num => num * 2)); // [2, 4, 6, 8, 10, 12, 14, 16, 18]

03 .forEach()

Array.forEach(callback)

  • 배열의 각 요소마다 한 번씩 제공된 함수를 실행합니다.
  • 반환 값이 없습니다.
[
    "apple",
    "banana",
    "cherry"
].forEach(fruit => console.log(fruit)); // apple, banana, cherry

04 .map()

Array.map(callback)

  • 순서는 원본 배열과 동일합니다.
[
    1,
    2,
    3
].map(num => num * 2); // [2, 4, 6]

07 .join()

Array.join(separator)

  • 배열의 모든 요소를 연결해 하나의 문자열로 만듭니다.
  • separator는 요소를 구분하는 문자열이며, 생략하면 쉼표가 기본값으로 사용됩니다.
[
    "apple",
    "banana",
    "cherry"
].join(", "); // "apple, banana, cherry"

09 .slice()

Array.slice(start, end)

  • 지정된 시작 인덱스부터, 또는 종료 인덱스 전까지의 요소로 새로운 배열을 반환합니다.
  • 원본 배열은 수정되지 않습니다.
[
    "apple",
    "banana",
    "cherry",
    "date"
].slice(1, 3); // ["banana", "cherry"]

10 .splice()

Array.splice(start, deleteCount, item1, item2, ...)

  • 배열에서 요소를 제거하고, 그 자리에 새 요소를 추가합니다.
  • 삭제한 요소를 포함한 배열을 반환합니다.
let fruits = ["apple", "banana", "cherry", "date"];
fruits.splice(2, 1, "orange"); // ["cherry"]
// fruits 배열은 이제 ["apple", "banana", "orange", "date"] 입니다.

11 .sort()

Array.sort(compareFunction)

  • 배열의 요소를 적절한 위치에 정렬하고 배열 자체를 반환합니다.
  • 기본적으로 문자열로 변환하여 유니코드 코드 포인트 순서로 정렬합니다.
[
    4,
    2,
    5,
    1,
    3
].sort((a, b) => a - b); // [1, 2, 3, 4, 5]

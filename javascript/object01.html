01. at()

"문자열".at(위치값)

  • 지정한 숫자(index)를 받아 , 문자열에서 해당되는 인덱스의 요소를 반환합니다.
  • 대소문자를 구별합니다.
  • 문자열의 길이보다 큰 인덱스를 지정하면 undefined가 발생합니다.
  • 음수인 경우 문자열의 오른쪽에서부터 인덱스를 처리합니다.
{
    "javascript reference".at()       //j
    "javascript reference".at(0)      //j
    "javascript reference".at(1)      //a
    "javascript reference".at(2)      //v
    "javascript reference".at(3)      //a
    "javascript reference".at(4)      //s
    "javascript reference".at(5)      //c
    "javascript reference".at(-1)     //e
    "javascript reference".at(-2)     //c
    "javascript reference".at(-3)     //n
    "javascript reference".at(-4)     //e
    "javascript reference".at(-5)     //r
    "javascript reference".at(100)    // undefined
}

06. includes()

"문자열".includes(찾을 문자열)

  • 지정한 문자열이 해당 문자열에 포함되어 있는지 확인합니다.
  • 대소문자를 구별합니다.
  • 문자열에 포함되어 있으면 true를 반환하고, 그렇지 않으면 false를 반환합니다.
{
    "javascript reference".includes("javascript")   // true
    "javascript reference".includes("reference")    // true
    "javascript reference".includes("script")       // false
    "javascript reference".includes("JavaScript")   // false
}

07. indexOf()

"문자열".indexOf(대상문자열)

  • 대상 문자열에서 첫 번째로 발견된 인덱스를 반환합니다.
  • 대소문자를 구별합니다.
  • 대상 문자열이 없으면 -1을 반환합니다.
{
                "javascript reference".indexOf("j")    //0
                "javascript reference".indexOf("a")    //1

                "javascript reference".indexOf("e")    //17
                "javascript reference".indexOf("n")    //14
                "javascript reference".indexOf("r")    //15

}

10. match()

"문자열".match(정규식)

  • 정규식과 일치하는 문자열을 반환합니다.
  • 처음으로 일치하는 부분만 반환하며, 전역 플래그(g)를 사용하면 모든 일치하는 부분을 반환합니다.
  • 대소문자를 구별합니다.
  • 일치하는 부분이 없으면 null을 반환합니다.
{
                "javascript reference".match(/j/)       //["j"]
                "javascript reference".match(/a/)       //["a"]
                "javascript reference".match(/v/)       //["v"]
                "javascript reference".match(/s/)       //["s"]
                "javascript reference".match(/c/)       //["c"]
                "javascript reference".match(/e/)       //["e"]
                "javascript reference".match(/n/)       //["n"]
                "javascript reference".match(/r/)       //["r"]
                "javascript reference".match(/x/)       //null
            }

16. replace()

"문자열".replace(찾을문자열 또는 정규식, 대체할문자열)

  • 첫 번째로 일치하는 부분을 대체할 문자열로 변경합니다.
  • 정규식을 사용하여 대체할 수 있습니다.
  • 대소문자를 구별합니다.
{
                "javascript reference".replace("j", "J")    //"Javascript reference"
                "javascript reference".replace(/a/g, "A")   //"jAvAscript reference"
                "javascript reference".replace("v", "")     //"javascript reference"
                "javascript reference".replace(/s/, "S")    //"javScript reference"
                "javascript reference".replace("x", "X")    //"javascript reference"
            }

19. slice()

"문자열".slice(시작인덱스, 종료인덱스)

  • 시작 인덱스부터 종료 인덱스까지의 부분 문자열을 반환합니다.
  • 종료 인덱스는 반환되는 문자열에 포함되지 않습니다.
  • 음수 값은 문자열의 끝에서부터의 거리를 나타냅니다.
  • 시작 인덱스가 종료 인덱스보다 큰 경우 빈 문자열을 반환합니다.
{
                "javascript reference".slice(0, 4)     //"java"
                "javascript reference".slice(4, 10)    //"script"
                "javascript reference".slice(-10)      //"reference"
                "javascript reference".slice(10, 4)    //""
            }

20. substring()

"문자열".substring(시작인덱스, 종료인덱스)

  • 시작 인덱스부터 종료 인덱스 전까지의 부분 문자열을 반환합니다.
  • 음수 값은 0으로 처리됩니다.
  • 종료 인덱스가 생략되면 문자열의 끝까지 반환됩니다.
  • 시작 인덱스가 종료 인덱스보다 큰 경우 두 인덱스가 교환되어 동작합니다.
{
                "javascript reference".substring(0, 4)     //"java"
                "javascript reference".substring(4, 10)    //"script"
                "javascript reference".substring(10)       //"reference"
                "javascript reference".substring(10, 4)    //"script"
            }

21. substr()

"문자열".substr(시작인덱스, 길이)

  • 시작 인덱스부터 지정한 길이만큼의 부분 문자열을 반환합니다.
  • 음수 값을 가질 경우 문자열의 끝에서부터의 거리로 처리됩니다.
  • 길이가 생략되면 시작 인덱스부터 문자열 끝까지 반환됩니다.
{
                "javascript reference".substr(0, 4)      //"java"
                "javascript reference".substr(4, 6)      //"script"
                "javascript reference".substr(10)        //"reference"
                "javascript reference".substr(-10)       //"reference"
            }

22. split()

"문자열".split(구분자)

  • 구분자를 기준으로 문자열을 나누어 배열로 반환합니다.
  • 구분자가 생략되면 문자열 전체를 단일 요소로 하는 배열을 반환합니다.
{
                "javascript reference".split(" ")      //["javascript", "reference"]
                "javascript,reference".split(",")      //["javascript", "reference"]
                "javascript reference".split("")       //["j", "a", "v", "a", "s", "c", "r", "i", "p", "t", " ", "r", "e", "f", "e", "r", "e", "n", "c", "e"]
}

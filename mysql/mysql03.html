DataBase 생성

create database 이름; 
mysql> create database test; // test라는 이름의 DB 생성.
Query OK, 1 row affected (0.00 sec) // DB 생성 완료

mysql> show databases; 
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |    
| performance_schema |
| sys                |
| test                |
+--------------------+
5 rows in set (0.00 sec)

DataBase 삭제

drop database 이름; 
mysql> drop database test; // test라는 이름을 가진 DB를 삭제한다.
